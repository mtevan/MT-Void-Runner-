# 🌌 MT Void Runner

[![Live Demo](https://img.shields.io/badge/Demo-Live%20On%20Vercel-brightgreen?style=for-the-badge)](https://mtvoidrunner.vercel.app/)
[![GitHub repo](https://img.shields.io/badge/GitHub-Repository-blue?style=for-the-badge&logo=github)](https://github.com/mtevan/MT-Void-Runner-)

**MT Void Runner** is an immersive, fast-paced web-based arcade game where players navigate a cosmic vessel or avatar through an endless, hazard-filled void. Avoid deep-space obstacles, pick up cosmic boosts, and test your reflexes to see how long you can survive the endless abyss.

🚀 **[Play MT Void Runner Now!](https://mtvoidrunner.vercel.app/)**

---

## 🎮 Gameplay & Features

- **Endless Void Generation:** Environments and obstacles change dynamically as you progress, offering a unique challenge with every run.
- **Increasing Difficulty:** The longer you survive, the faster and more intense the void becomes.
- **Responsive Controls:** Fine-tuned for both precision and fluid movement.
- **Instant Web Play:** Fully optimized and instantly accessible directly from your browser via Vercel.

## 🛠️ Tech Stack

This project leverages modern web development tools for high performance and smooth rendering:

- **Frontend:** HTML5, CSS3, JavaScript / TypeScript *(e.g., React, Vue, or Vanilla JS)*
- **Graphics/Engine:** Canvas API *(or Three.js / PixiJS if applicable)*
- **Deployment:** [Vercel](https://vercel.com)
